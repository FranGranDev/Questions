В проекте есть 4 тестовые сцены: Один ответ Single, выбор ответов Multi, совмещение картинки Image и все типы по порядку AllScene

В скрипте Main(находится в QuestionCanvas) присутсвует массив Question Data в которой
вы можете настраивать свои вопросы, а именно:
-Тип теста(Один ответ Single, выбор ответов Multi, совмещение картинки Image)
-Для Single и Multi текст вопроса и верный ли он.
-Для Image ссылка на объект со скриптом совмещения

-Настройка картинки для совмещения текста(пример находится в папке Prefabs под именем ImagePlane)

На объекте находится скрипт ImageTaken где есть массив ImageItem в котором задается индекс, текст,
который надо будет совместить с картинкой и ссылки на текст кнопок(TMPro),
которые нужно заполнить соответственными компонентами, которые находятся в Кнопках.

В дочерних объектах картинки должны находится столько Кнопок, сколько элементов массива ImageItem.
В дочернем объекте кнопки должен находтся тескт(TMPro), на который надо сделать ссылку в ImageItem.
На кнопке есть компонент Event Trigger, для него нужно настроить индекс функции OnPointerEnter,
индекс должен соответствовать индексу этой кнопки в массиве ImageItem.

-Префабы кнопок ответов находятся в папке Prefabs.

-В функции SetQuestions() есть функция RandomizePosition(), которая рандомно меняет позицию ответов в списке.
